C - Strings

Time limit per test: 1 seconds

Memory limit per test: 1 GB

Given some words, print them back out in reverse order

Input

1 <= n <= 100, the number of words to read in. What follows is n lines, each containing 1 word.

Output

n words in reverse order of input
